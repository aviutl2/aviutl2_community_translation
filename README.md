# AviUtl2 Community Translation

メンテナー：[@sevenc-nanashi](https://github.com/sevenc-nanashi)

[English README](README.en.md)

AviUtl2の非公式多言語翻訳プロジェクト。
このリポジトリでは、AviUtl2本体および付属するエフェクトの翻訳ファイルを管理しています。

## 翻訳の仕方

> [!NOTE]
> そのうちCrowdinなどの翻訳プラットフォームなどに移行する予定です。

1. このリポジトリをフォークします。
2. `translations/`ディレクトリ内の翻訳したい言語のフォルダに移動します。
3. 翻訳したいファイルを編集します。
4. 変更をコミットし、プルリクエストを作成します。

> [!TIP]
> `editor.yml`にはAviUtl2本体の翻訳が含まれています。
> `effects.yml`には付属エフェクトの翻訳が含まれています。
